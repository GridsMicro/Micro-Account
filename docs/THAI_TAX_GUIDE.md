## 2. แบบฟอร์มภาษีเงินได้ (ภ.ง.ด.)

### 🧾 ภ.ง.ด. 3 - หัก ณ ที่จ่าย (บุคคลธรรมดา)

**ใช้เมื่อ:** บริษัทจ่ายเงินให้บุคคลธรรมดา เช่น
- ค่าจ้างฟรีแลนซ์
- ค่าบริการ outsource
- ค่าจ้างเหมา
- ค่านายหน้า

**อัตราหักภาษี:**
| ประเภทรายได้ | อัตราภาษี |
|-------------|----------|
| ค่าบริการทั่วไป | **3%** |
| ค่านายหน้า | 5% |
| ค่าจ้างเหมา | 3% |
| ค่าเช่าทรัพย์สิน | 5% |
| ดอกเบี้ย | 15% |
| เงินปันผล | 10% |

**การบันทึกบัญชี:**
```
เดบิต: ค่าใช้จ่าย (ยอดเต็ม)
เครดิต: เงินสด/ธนาคาร (ยอดหลังหักภาษี)
เครดิต: ภาษีหัก ณ ที่จ่าย 3% (ภงด.3)
```

---

### 🏢 ภ.ง.ด. 50 - ภาษีเงินได้นิติบุคคล (ปี)

**ใช้เมื่อ:** สิ้นรอบบัญชี (ปีงบประมาณ หรือ 12 เดือน)

**การคำนวณ:**
```
กำไรสุทธิ = รายได้ - ค่าใช้จ่าย - ค่าเสื่อม - ขาดทุนสะสม
ภาษี = กำไรสุทธิ × อัตราภาษีนิติบุคคล (20%)
```

**อัตราภาษีนิติบุคคล (ปี 2566-2567):**
| กำไรสุทธิ | อัตราภาษี |
|----------|----------|
| 0 - 300,000 บาท | **ยกเว้น** |
| 300,001 - 3,000,000 บาท | 15% |
| 3,000,001 ขึ้นไป | 20% |

> 💡 **หมายเหตุ:** ธุรกิจขนาดกลางและเล็ก (SME) ที่มีรายได้ไม่เกิน 30 ล้านบาท ได้รับสิทธิอัตราภาษีพิเศษ

---

### 📅 ภ.ง.ด. 51 - ภาษีเงินได้นิติบุคคลครึ่งปี

**ใช้เมื่อ:** ยื่นภาษีกลางปี (ภายใน 2 เดือนจากวันครบรอบ 6 เดือนของรอบบัญชี)

**การคำนวณ:**
```
ประมาณกำไรครึ่งปี = กำไร 6 เดือนแรก × 2
ภาษีครึ่งปี = ประมาณกำไร × อัตราภาษี × 50%
```

**ยกตัวอย่าง:**
- รอบบัญชี: ม.ค. - ธ.ค.
- ยื่น ภ.ง.ด. 51: ภายในสิ้นเดือน ส.ค. (2 เดือนจากสิ้นเดือน มิ.ย.)

---

### 📄 ภ.ง.ด. 53 - หัก ณ ที่จ่าย (นิติบุคคล)

**ใช้เมื่อ:** บริษัทจ่ายเงินให้บริษัท/นิติบุคคลอื่น

**กรณียอดนิยม:**
| ประเภท | อัตรา | รหัสมาตรา |
|--------|-------|----------|
| ค่า License Software | **5%** | ม.40(3) |
| ค่าลิขสิทธิ์ | 5% | ม.40(3) |
| ค่าแฟรนไชส์ | 5% | ม.40(3) |
| ค่าเช่าอสังหา | 5% | ม.40(5) |
| ดอกเบี้ย | 1% | ม.40(4) |

> ⚠️ **สำคัญมาก:** การขาย Software License ให้บริษัทในประเทศไทย ลูกค้าจะหักภาษี **5%** โดยอัตโนมัติ

## 3. แบบฟอร์มภาษีมูลค่าเพิ่ม (ภ.พ.)

### 🧮 ภ.พ. 30 - VAT 7% (ในประเทศ)

**ความหมาย:** รายงานภาษีมูลค่าเพิ่มประจำเดือน

**การคำนวณ:**
```
Output VAT (ภาษีขาย) = ยอดขาย × 7%
Input VAT (ภาษีซื้อ) = ยอดซื้อ × 7%
VAT สุทธิ = Output VAT - Input VAT

ถ้า Output > Input = ต้องจ่ายสรรพากร
ถ้า Output < Input = ขอคืนหรือทบยอดไปเดือนหน้า
```

**ตัวอย่าง:**
```
ยอดขายเดือนนี้: 100,000 บาท
Output VAT: 100,000 × 7% = 7,000 บาท

ยอดซื้อเดือนนี้: 60,000 บาท
Input VAT: 60,000 × 7% = 4,200 บาท

VAT ที่ต้องจ่าย: 7,000 - 4,200 = 2,800 บาท
```

**การบันทึกบัญชีใน Micro-Account:**
```
ตอนขาย:
เดบิต: ลูกหนี้ 107,000
เครดิต: รายได้ขาย 100,000
เครดิต: ภาษีขาย 7,000

ตอนซื้อ:
เดบิต: สินค้า/ค่าใช้จ่าย 60,000
เดบิต: ภาษีซื้อ 4,200
เครดิต: เจ้าหนี้ 64,200

ตอนจ่าย VAT:
เดบิต: ภาษีขาย 7,000
เครดิต: ภาษีซื้อ 4,200
เครดิต: เงินสด 2,800
```

**กำหนดยื่น:** ภายในวันที่ 15 ของเดือนถัดไป

---

### 🌍 ภ.พ. 36 - VAT บริการต่างประเทศ

**ใช้เมื่อ:** รับบริการจากต่างประเทศ (เช่น AWS, Google, Facebook Ads)

**การคำนวณ Reverse Charge:**
```
ค่าบริการ: $1,000 USD
อัตราแลกเปลี่ยน: 35 บาท/USD
มูลค่าบาท: 35,000 บาท

VAT 7%: 35,000 × 7% = 2,450 บาท
(นำเข้าเป็นค่าใช้จ่ายได้เต็ม 37,450 บาท)
```

**การบันทึกบัญชี:**
```
เดบิต: ค่าใช้จ่าย 35,000
เดบิต: ค่าใช้จ่าย (VAT) 2,450
เครดิต: ภาษีซื้อ 2,450
เครดิต: เงินสด/ธนาคาร 35,000
```

> 💡 **เคล็ดลับ:** ภ.พ.36 สามารถนำ VAT ไปขอคืนหรือหักออกจาก VAT ขายได้เหมือนกัน

## 4. ภาษีหัก ณ ที่จ่าย (Withholding Tax)

### 💰 ความหมายของ WHT 3%

**WHT 3%** = ภาษีที่ผู้จ่ายเงินหักไว้ 3% จากค่าบริการ เพื่อนำส่งสรรพากรแทนผู้รับเงิน

**อัตราภาษีหัก ณ ที่จ่ายที่ใช้บ่อย:**

| ประเภท | อัตรา | แบบฟอร์ม |
|--------|-------|---------|
| ค่าบริการทั่วไป | **3%** | ภ.ง.ด. 3 หรือ ภ.ง.ด. 53 |
| ค่า License Software | **5%** | ภ.ง.ด. 53 |
| ค่าลิขสิทธิ์ | 5% | ภ.ง.ด. 53 |
| ค่าแฟรนไชส์ | 5% | ภ.ง.ด. 53 |
| ค่าเช่าอสังหา | 5% | ภ.ง.ด. 3/53 |
| ค่าจ้างเหมา | 3% | ภ.ง.ด. 3 |
| ดอกเบี้ย | 1% | ภ.ง.ด. 3 |

---

### 📋 หนังสือรับรองการหักภาษี ณ ที่จ่าย

**ความสำคัญ:** เป็นหลักฐานที่ผู้ถูกหักต้องใช้ยื่นภาษีเพื่อขอเครดิตภาษีคืน

**องค์ประกอบ:**
1. เลขที่เอกสาร
2. ชื่อผู้ถูกหักและผู้หัก
3. Tax ID ทั้งสองฝ่าย
4. ประเภทเงินได้และมูลค่า
5. จำนวนภาษีที่หัก
6. วันที่หักและวันที่จ่าย

---

## 5. การบันทึกบัญชี WHT 3% ในระบบ

### 🔄 กรณีที่ 1: เราถูกลูกค้าหัก 3% (ถูกหักจากเงินที่จะได้รับ)

**สถานการณ์:** เราขายบริการ 100,000 บาท ลูกค้าหัก WHT 3% = 3,000 บาท
เราได้รับเงินจริง = 97,000 บาท

**การคำนวณ:**
```
ค่าบริการ: 100,000 บาท
WHT 3%: 100,000 × 3% = 3,000 บาท
ได้รับเงินสุทธิ: 100,000 - 3,000 = 97,000 บาท
```

**การบันทึกบัญชี:**
```
ขั้นตอนที่ 1 - ตอนออกใบแจ้งหนี้:
เดบิต: ลูกหนี้ 100,000
เครดิต: รายได้ 100,000

ขั้นตอนที่ 2 - ตอนรับชำระ (ถูกหัก 3%):
เดบิต: เงินฝากธนาคาร 97,000
เดบิต: ภาษีเงินได้ถูกหัก ณ ที่จ่าย 3,000  ⭐ สินทรัพย์
เครดิต: ลูกหนี้ 100,000
```

**บันทึกใน Micro-Account:**
- ไปที่เมนู **รับชำระเงิน** (`/payments/new`)
- กรอกยอดเงิน: 100,000
- เลือก **หัก ณ ที่จ่าย 3%**
- ระบบจะคำนวณให้อัตโนมัติ:
  - ยอดรับสุทธิ: 97,000
  - WHT: 3,000 (บันทึกเป็นทรัพย์สิน)

**ตอนสิ้นปี:**
```
นำ WHT 3,000 บาท ไปหักออกจากภาษีเงินได้ที่ต้องจ่าย
หรือขอคืนกรมสรรพากรถ้าภาษีไม่ถึง
```

---

### 🔄 กรณีที่ 2: เราหัก 3% จากเอาท์ซอร์ส/ผู้รับเหมา

**สถานการณ์:** จ้างเอาท์ซอร์สทำงาน 50,000 บาท เราหัก WHT 3% = 1,500 บาท
เราจ่ายเงินจริง = 48,500 บาท

**การคำนวณ:**
```
ค่าจ้าง: 50,000 บาท
WHT 3%: 50,000 × 3% = 1,500 บาท
จ่ายเงินสุทธิ: 50,000 - 1,500 = 48,500 บาท
```

**การบันทึกบัญชี:**
```
ขั้นตอนที่ 1 - ตอนบันทึกค่าใช้จ่าย:
เดบิต: ค่าจ้างเหมา 50,000
เครดิต: เจ้าหนี้ 50,000

ขั้นตอนที่ 2 - ตอนจ่ายเงิน (หัก 3%):
เดบิต: เจ้าหนี้ 50,000
เครดิต: เงินฝากธนาคาร 48,500
เครดิต: ภาษีหัก ณ ที่จ่าย 3% 1,500  ⭐ หนี้สินต่อสรรพากร
```

**บันทึกใน Micro-Account:**
- ไปที่เมนู **จ่ายเงิน/บันทึกค่าใช้จ่าย** (`/expenses` หรือ `/vouchers`)
- กรอกยอดเงิน: 50,000
- เลือก **หักภาษี ณ ที่จ่าย 3%**
- ระบบจะลงบัญชีให้อัตโนมัติ

**สิ่งที่ต้องทำต่อ:**
1. **ออกหนังสือรับรองหัก ณ ที่จ่าย** ให้ผู้รับเหมา
2. **ยื่น ภ.ง.ด. 3** ให้สรรพากรภายในวันที่ 15 เดือนถัดไป
3. **นำส่งเงินภาษี** 1,500 บาท ให้สรรพากร

## 6. ตัวอย่างการคำนวณ

### 📊 ตัวอย่างที่ 1: ขาย Software License (ถูกหัก 5%)

**ข้อมูล:**
- ขาย License: 100,000 บาท
- VAT 7%: 7,000 บาท
- รวมใบแจ้งหนี้: 107,000 บาท
- ลูกค้าหัก WHT 5%: 5,000 บาท (จาก 100,000)

**การคำนวณ:**
```
ยอดที่ต้องได้รับ: 107,000 - 5,000 = 102,000 บาท
```

**การบันทึกบัญชี:**
```
1. ออกใบแจ้งหนี้:
   เดบิต: ลูกหนี้ 107,000
   เครดิต: รายได้ License 100,000
   เครดิต: ภาษีขาย 7,000

2. รับชำระ (ถูกหัก 5%):
   เดบิต: ธนาคาร 102,000
   เดบิต: ภาษีถูกหัก 5% 5,000
   เครดิต: ลูกหนี้ 107,000
```

**บันทึกใน Micro-Account:**
- ระบบจะรองรับ WHT 5% สำหรับ License
- บันทึก WHT เป็นสินทรัพย์ (ภาษีเงินได้ถูกหัก ณ ที่จ่าย)

---

### 📊 ตัวอย่างที่ 2: จ้าง Freelance (หัก 3%)

**ข้อมูล:**
- จ้างทำเว็บไซต์: 30,000 บาท
- หัก WHT 3%: 900 บาท
- จ่ายสุทธิ: 29,100 บาท

**การคำนวณ:**
```
ค่าจ้าง: 30,000
WHT 3%: 30,000 × 0.03 = 900
จ่ายจริง: 29,100
```

**การบันทึกบัญชี:**
```
1. บันทึกค่าใช้จ่าย:
   เดบิต: ค่าจ้างเหมา 30,000
   เครดิต: เจ้าหนี้ Freelance 30,000

2. จ่ายเงิน (หัก 3%):
   เดบิต: เจ้าหนี้ Freelance 30,000
   เครดิต: ธนาคาร 29,100
   เครดิต: ภาษีหัก ณ ที่จ่าย 3% 900
```

---

### 📊 ตัวอย่างที่ 3: VAT และ WHT รวมกัน

**สถานการณ์:** ขายบริการ Consult 50,000 + VAT 7% = 53,500
ลูกค้าหัก WHT 3% จาก 50,000 = 1,500

**การคำนวณ:**
```
ราคาบริการ: 50,000
VAT 7%: 3,500
รวม: 53,500

หัก WHT 3%: 1,500
สุทธิที่ได้รับ: 53,500 - 1,500 = 52,000
```

**การบันทึกบัญชี:**
```
1. ขาย:
   เดบิต: ลูกหนี้ 53,500
   เครดิต: รายได้ 50,000
   เครดิต: ภาษีขาย 3,500

2. รับเงิน:
   เดบิต: ธนาคาร 52,000
   เดบิต: ภาษีถูกหัก 3% 1,500
   เครดิต: ลูกหนี้ 53,500
```
